# Cumpleaños de Lore (28)

Página de cumpleaños con una cronología, tres canciones y una recompensa final.

## Añadir las fotos

Guarda las tres imágenes dentro de `assets/images/` con estos nombres exactos:

- `foto1.jpg`
- `foto2.jpg`
- `foto3.jpg`

Si una imagen todavía no está disponible, la página muestra un aviso en su lugar. Se
recomienda comprimir cada foto antes de subirla (idealmente por debajo de 1 MB).

## Música

Las canciones se reproducen mediante los reproductores oficiales de Spotify, por lo
que no es necesario subir archivos de audio al repositorio:

- Demons — Imagine Dragons
- Secrets — OneRepublic
- Creo en Ti — Reik

## Publicar con GitHub Pages

1. Sube los cambios a la rama `main` del repositorio.
2. En GitHub, abre **Settings > Pages**.
3. En **Build and deployment**, selecciona **Deploy from a branch**.
4. Elige la rama `main`, la carpeta `/ (root)` y pulsa **Save**.

GitHub mostrará la dirección pública cuando termine el despliegue.
